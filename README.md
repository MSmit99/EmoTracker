# EmoTracker
A multi-modal emotion recognition system using facial expressions, speech, and text sentiment to classify six core emotions. Includes an interactive dashboard for video or live input, showing real-time emotion probabilities. Designed to be accurate, accessible, and privacy-friendly.
---
To properly run the application, you must install the TER model from google drive, and place it in the models directory. The model can be found here: https://drive.google.com/file/d/1DnGZ0URUy2VR8hKUbWg89EfHPOLRQi-f/view?usp=sharing
